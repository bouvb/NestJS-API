# NestJS-API

Quick NestJS web API to test the framework
[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository based

## Project setup

### Install

```bash
$ pnpm install
```

### Compile and run the project

```bash
# development
$ pnpm run start

# watch mode to see modification without rerun the app
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

### Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
